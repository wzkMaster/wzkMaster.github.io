## 0. 计算机科学入门

如果你只是想了解一下怎么做一个网页或者小程序，可以跳过这一节，直接从 `HTML` 开始。但如果想要深入地学习前端和编程，那么必须先从这一步开始，学习一些编程的入门知识。

## 1. HTML

HTML用于表示网页的内容，包括文字、图像、视频等。

## 2. CSS

CSS用于改变HTML中的元素样式，让网页更加漂亮。

## 3. JavaScript

前端工程师大部分的时间都是在和JavaScript打交道，它可以让网页能够与用户动态交互。
